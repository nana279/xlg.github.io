新手入门（基础操作 + 必备道具）

基础操作

  • 核心移动：W/A/S/D 或方向键控制角色移动，鼠标点击交互（采集、对话、放置物品）；
  • 工具使用：左键使用工具（斧头砍树、镐子挖矿、锄头耕地、洒水壶浇水），右键放置物品 / 打开箱子；
  • 时间管理：游戏内 1 天 = 现实 14 分钟（6:00-22:00 可活动，22:00 后不睡觉会晕倒扣钱）；
  • 体力值：行动消耗体力，吃料理 / 水果可恢复，体力为 0 会晕倒。

必备道具

  • 初期必解锁：工具箱（砍树挖矿解锁，可存放工具 / 物品）、背包扩容（找皮埃尔杂货店花金币升级，优先升 2 级）；
  • 核心工具：铜镐（挖矿效率提升，优先升级）、铁斧头（砍大树更快）、洒水壶（升级后单次浇水范围更大）；
  • 随身必备：野莓 / 韭葱（初期恢复体力的免费食材）、纤维（制作稻草人防乌鸦偷菜）、种子（优先种防风草，生长快收益稳）。

农场规划（不同农场类型对比表 + 布局参考）

不同农场类型对比

农场类型 核心优势 劣势 适合人群
标准农场 耕地面积最大，无地形限制 无特殊资源加成 新手、专注种植/畜牧
河边农场 钓鱼收益高，河边可直接钓 耕地面积仅标准的 1/3 喜欢钓鱼的玩家
森林农场 每天刷新硬木，蘑菇树多 耕地分散且面积小 喜欢采集/制作的玩家
山顶农场 矿区近，宝石 / 矿石产量高 耕地少且地形崎岖 专注挖矿 / 锻造的玩家
荒野农场 可刷怪物掉落稀有材料 夜间会刷怪，新手易翻车 有一定经验的硬核玩家

布局参考

  • 前期（1-5 天）：左上角划分 10×10 耕地种防风草，右侧留 2×3 区域养 2 只鸡（解锁鸡舍后）；
  • 中期（10 天后）：中间区域建洒水器（优先做铱金洒水器），周围种啤酒花/蔓越莓（持续收获）；
  • 后期：左下角建畜牧区（牛/羊），右下角建温室（四季种植稀有作物），留 1 格宽的小路方便移动。

通用技巧

  • 每天可送 1 次礼物，生日当天送礼好感 ×8（优先给核心 NPC 送最爱礼物）；
  • 所有 NPC 都喜欢 “七彩碎片”“黄金南瓜”（稀有，留着给关键 NPC）；
  • 避免送 “垃圾、史莱姆泥、鱼饵”（几乎所有 NPC 都讨厌）。

钓鱼小技巧

  • 钓鱼条绿色区域越小，鱼越稀有；按住鼠标左键让浮标跟着鱼移动，别出绿区；
  • 雨天钓鱼概率更高，稀有鱼更容易出；
  • 升级鱼竿（玻璃纤维竿→铱金鱼竿）可解锁鱼饵 / 浮标，提升上钩率。

彩蛋合集

  1. 春季第 15 天的花舞节，和好感度≥4 心的 NPC 跳舞，好感直接 + 2 心；
  2. 敲石头时站在石头正下方 / 正右方，可一次敲出 2 个矿石；
  3. 垃圾桶里能捡到皮埃尔杂货店的打折券（每天看一眼，省金币）；
  4. 给农场的狗 / 猫每天点一下头，会慢慢提升好感，后期会帮你捡道具；
  5. 用锄头挖农场的 “发光点”，大概率出硬木 / 金块 / 稀有种子；
  6. 冬季砸开结冰的湖面，能钓到冬季专属的 “冰鱼”，售价翻倍；
  7. 把钻石放在克林特的熔炉里，能熔成 “钻石戒指”（送礼神器，部分 NPC 最爱）。
